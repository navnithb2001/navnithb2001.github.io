# Navnith Bharadwaj — Portfolio

Personal portfolio website: [navnithb2001.github.io](https://navnithb2001.github.io)

Single-page site built with vanilla HTML, CSS, and JavaScript — no frameworks, no build step.

## Features

- Dark / light theme toggle (persisted in localStorage)
- Three accent palettes: cyan + emerald, amber + red, indigo + teal
- Particle canvas background, typed-text hero, scroll-reveal animations
- Fully responsive, respects `prefers-reduced-motion`

## Run locally

```sh
python3 -m http.server
# open http://localhost:8000
```
